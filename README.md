# Student Marks Prediction using AI

This project is a simple AI model built using Linear Regression.
It predicts student marks based on the number of hours studied.

## Tools Used
- Python
- Google Colab
- Scikit-learn
- Matplotlib
 
## Note
Detailed explanation of the code and output is provided inside
the Google Colab notebook (AI_Student_Marks_Prediction.ipynb file).
